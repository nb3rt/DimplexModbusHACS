# Status register values

The following table lists the status register values and their meanings for each software version.
All descriptions are provided in English.

| Register value | L/M software | H/J software |
| --- | --- | --- |
| 0 | Off | Off |
| 1 | Off | Heat pump on – heating |
| 2 | Heating | Heat pump on – heating |
| 3 | Swimming pool | Heat pump on – swimming pool |
| 4 | Domestic hot water | Heat pump on – domestic hot water |
| 5 | Cooling | Heat pump on – heating + 2nd heat generator |
| 6 | Reserved | Heat pump on – swimming pool + 2nd heat generator |
| 7 | Reserved | Heat pump on – domestic hot water + 2nd heat generator |
| 8 | Reserved | Primary pump pre-run |
| 9 | Reserved | Heating purge |
| 10 | Defrost | Lock (see lock register for J software) |
| 11 | Flow monitoring | Lower operating limit |
| 12 | Reserved | Low-pressure limit |
| 13 | Reserved | Low-pressure shutdown |
| 14 | Reserved | High-pressure safety |
| 15 | Reserved | Anti short-cycling lock |
| 16 | Reserved | Minimum standstill time |
| 17 | Reserved | Grid load limitation |
| 18 | Reserved | Flow monitoring |
| 19 | Reserved | Second heat generator active |
| 20 | Reserved | Low-pressure (brine/source) |
| 21 | Reserved | Heat pump on – defrost |
| 22 | Reserved | Upper operating limit |
| 23 | Reserved | External lock |
| 24 | Operating mode switch delay | Cooling operating mode |
| 25 | Reserved | Frost protection (cooling) |
| 26 | Reserved | Flow temperature limit |
| 27 | Reserved | Dew point monitor |
| 28 | Reserved | Dew point |
| 29 | Passive cooling | Reserved |
| 30 | Lock (see lock register) | Reserved |
