The following table lists the lock register values and their meanings for each software version.
All descriptions are provided in English.

| Register value | L/M software | J software | H software |
| --- | --- | --- | --- |
| 0 | No lock | No lock | No lock |
| 1 | High temperature operating limit | Outdoor temperature | Reserved |
| 2 | Volume flow | Heat pump operating limit | Bivalent alternative |
| 3 | Reserved | Regenerative | Bivalent regenerative |
| 4 | Reserved | Reserved | Return temperature |
| 5 | Function check | Domestic hot water post-heating | Domestic hot water |
| 6 | High temperature operating limit | System check | System check |
| 7 | System check | Utility (EVU) lockout | Utility (EVU) lockout |
| 8 | Delay switching to cooling | Reserved | Reserved |
| 9 | Pump pre-run | High pressure | Reserved |
| 10 | Minimum standstill time | Low pressure | Reserved |
| 11 | Grid load limitation | Flow | Reserved |
| 12 | Anti short-cycling lock | Soft starter | Reserved |
| 13 | Domestic hot water post-heating | Reserved | Reserved |
| 14 | Regenerative | Reserved | Reserved |
| 15 | Utility (EVU) lockout | Reserved | Reserved |
| 16 | Soft starter | Reserved | Reserved |
| 17 | Flow | Reserved | Reserved |
| 18 | Heat pump operating limit | Reserved | Reserved |
| 19 | High pressure | Reserved | Reserved |
| 20 | Low pressure | Reserved | Reserved |
| 21 | Heat source operating limit | Reserved | Reserved |
| 23 | System limit | Reserved | Reserved |
| 24 | Primary circuit load | Reserved | Reserved |
| 25 | External lock | Reserved | Reserved |
| 29 | Inverter | Reserved | Reserved |
| 31 | Warm-up | Reserved | Reserved |
| 33 | EvD initialization | Reserved | Reserved |
| 34 | Second heat generator enabled | Reserved | Reserved |
